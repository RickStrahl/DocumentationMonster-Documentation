Documentation Monster's primary output is Html and an Html based static Web site. Output generation and publishing starts with building the Project to a static Web Site and then choosing one of the publishing options.

The easiest way to do this is to Build dialog, and then Publish from there, which you can do by using the @icon-duotone-screwdriver-wrench **Build Project Web Site Html** Toolbar icon or **Build -> Build Project Web Site Html** from the Menu, both of which [build a Static Web Site](dm-topic://_3aiujpecyp) and then takes you to the [Publish Dialog](dm-topic://_75ru5oj7g9) with various publishing options.
